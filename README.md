# Orna

# Virtual Jewelry Try-On Platform

Orna is a web-based application that allows users to virtually try on jewelry using computer vision and facial landmark detection.

# Features

- Upload face image
- Real-time face detection (coming soon)
- Virtual jewelry overlay (coming soon)

# Tech Stack

- React
- JavaScript
- MediaPipe (planned)

# Goal of this project

To build an interactive virtual try-on system that improves online jewelry shopping experience using AI and computer vision.
